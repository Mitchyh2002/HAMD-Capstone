The Finalised Product Can be found in the Finalised Prototype Folder.

To Run the prototype please do the following:

	Open Prototype/Backend/Program/__init__.py
	On line 45 change the app.config['SQLALCHEMY_DATABASE_URI'] variable to a string containing your local database.
	Once this is done please, run the Prototype/Backend/create_db.py to build the database.
	Once done please run Protype/run.bat this will start the react application in one command line & install all required python modules and start the flask application in another command line. 
	Once done a browser will open the react script for the application.
	If you would like to see unit tests please see Prototype/Backend/UnitTest.py
